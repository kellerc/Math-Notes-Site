Exponential Functions -
Any function in the form f(x)=bx where b>0, b≠1 is called an exponential function

f(x) = bx -
Most important base in advanced math is the irrational # 'e'
e ≈ 2.718
e is defined as e = limit (1+1/n)n
n --> ∞

n|(1+1/n)n
----------------
1|(1+1/1)1 = 2
10|(1+1/10)10 = 2.59374
100|(1+1/100)100 = 2.70481
1,000|(1+1/1,000)1,000 = 2.716923
10,000|(1+1/10,000)10,000 = 2.71814
100,000|(1+1/100,000)100,000 = 2.71826
1,000,000|(1+1/1,000,000)1,000,000 = 2.71828


Compound Interest -
A = P(1+r/n)nt
P = Principle (original)
n = Number of times per year compounded
r = Rate in decimal
t = Number of years


Continuous Compound Interest -
A = Pert
P = Principle
r = Rate in decimal
